## Users

Sales operators and small-business owners who run WhatsApp as their primary
CRM channel. They manage multiple WhatsApp instances via Evolution API,
capture leads from inbound messages, qualify them with AI scoring, and move
them through a pipeline (new → negotiation → qualified → lost). Their day
revolves around three surfaces: the inbox for real-time conversations, the
lead board for pipeline management, and analytics for performance tracking.

## Product Purpose

Leadflow turns WhatsApp into a structured sales pipeline. It connects to
Evolution API instances, captures every inbound message as a lead, scores
them with AI sentiment analysis, and presents the whole operation through a
cinematic dark-canvas interface. The product is a tool first — it needs to
feel precise, fast, and trustworthy, not decorative.

## Brand & Tone

- **Brand voice**: confident, technical, quietly premium. Think observability
  dashboard meets editorial type — no marketing fluff, no emoji-laden CTAs.
- **Tone**: serious but not sterile. Portuguese UI copy. Labels are terse
  and uppercase; body copy is sparing and purposeful.
- **Anti-references**: not a SaaS cream-puff, not a neon-crypto dashboard,
  not a healthcare-white-clean. The dark void is non-negotiable; accents are
  atmospheric, not decorative.

## Strategic Principles

1. **Signal over chrome.** Every surface should reduce cognitive load. If it
   doesn't help the operator decide, it goes.
2. **Type carries hierarchy.** The Resend-inspired type system (display serif
   for section openers, Inter for UI) is the primary visual language — not
   color, not icons.
3. **Dark canvas, precise borders.** Pure black background, frost borders for
   structure, atmospheric glows only for section emphasis.
4. **AI is a feature, not the product.** The AI indicators exist but don't
   dominate. The operator's data is the star.

## Key Surfaces

1. **Dashboard** — system status, KPI cards, recent leads, AI readiness CTA
2. **Leads** — Kanban board + table view, drag-and-drop pipeline management
3. **Inbox** — WhatsApp chat interface, AI insights sidebar, contact list
4. **Analytics** — 18-week trend chart, conversion metrics, export
5. **Settings** — instance management, API configuration, user preferences
